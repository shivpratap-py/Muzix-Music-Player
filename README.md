# 🎵 Muzix

A modern Spotify-inspired music player built with HTML, CSS, and JavaScript.

🌐 **Live Demo:** https://muzixbyshiv.netlify.app/

## Overview

Muzix is a responsive web-based music player that allows users to browse albums, view playlists, and enjoy smooth music playback directly in the browser. The application dynamically loads albums and tracks, provides playback controls, volume management, and an intuitive user experience inspired by modern music streaming platforms.

## Features

* 🎶 Dynamic album and playlist loading
* ⏯️ Play, Pause, Next & Previous controls
* 📊 Interactive seek bar with progress tracking
* 🔊 Volume adjustment and mute functionality
* ⌨️ Keyboard shortcut support
* 📱 Mobile-responsive design
* 🎨 Clean Spotify-inspired interface
* 🖼️ Album artwork support
* ⚡ Lightweight and fast

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* JSON
