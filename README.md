# 🎵 Muzix

Muzix is a modern Spotify-inspired music player built using vanilla HTML, CSS, and JavaScript. It delivers a smooth music listening experience with dynamic playlist loading, album management, playback controls, and a responsive user interface optimized for both desktop and mobile devices.

## ✨ Features

### 🎶 Music Playback

* Play and pause songs
* Previous and next track navigation
* Interactive seek bar
* Real-time song progress tracking
* Keyboard support (Spacebar to play/pause)

### 📂 Dynamic Playlists

* Load albums dynamically from JSON data
* Browse multiple music collections
* Album cover artwork support
* Playlist metadata and descriptions

### 🔊 Audio Controls

* Volume adjustment slider
* Mute and unmute functionality
* Live playback status updates

### 📱 Responsive Design

* Mobile-friendly interface
* Sliding sidebar navigation
* Hamburger menu for smaller screens
* Optimized layouts for desktop, tablet, and mobile devices

### 🎨 User Interface

* Spotify-inspired design
* Smooth animations and transitions
* Custom scrollbar styling
* Modern dark theme

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/muzix.git
```

### Open the Project

```bash
cd muzix
```

Run the project using a local server.

Examples:

```bash
npx serve .
```

or

```bash
python -m http.server
```

Then open:

```text
http://localhost:8000
```
